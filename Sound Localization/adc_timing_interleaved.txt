================================================================
NAPOMENA ZA TDOA IZRAČUN
================================================================

  Isti indeks s NE znači isti trenutak za različite kanale!
  CH2 je uvijek 0.9 µs kasniji od CH1 u istom uzorku s.

  Primjer: mic1 i mic2 oba triggeriraju na uzorku s = 5:
      t_mic1 = 4 × 62.5 + 0 × 0.9 =  250.0 µs
      t_mic2 = 4 × 62.5 + 1 × 0.9 =  250.9 µs
      TDOA   = 250.0 − 250.9       =   −0.9 µs  → mic2 čuje prvi

  Opća formula TDOA između mic1 (CH1) i mic2 (CH2):
      TDOA = (n1 − n2) × 62.5 µs − 0.9 µs

  Sanity check — fizikalni maksimum pri d = 5 cm:
      TDOA_max = 0.05 / 343 = 145.8 µs  →  max |n1 − n2| = 2
      Ako |n1 − n2| > 2 → lažni trigger, odbaci frame

================================================================
MAKSIMALNA RAZLUČIVOST KUTEVA — ANALIZA
Dodano: 2026-04-02
================================================================

Parametri:
  d    = 0.05 m     (razmak mikrofona)
  c    = 343 m/s    (brzina zvuka pri sobnoj temperaturi)
  fs   = 16000 Hz   (frekvencija uzorkovanja)
  T    = 62.5 µs    (period uzorkovanja = 1/fs)
  T_ch = 0.9 µs     (ADC channel offset CH1→CH2)

----------------------------------------------------------------
1. FIZIKALNI MAKSIMUM TDOA
----------------------------------------------------------------

  Maksimalni TDOA nastaje kada zvuk dolazi TOČNO uzduž osi
  mikrofona (φ = ±90°) — prolazi prvo kroz jedan mikrofon,
  a drugi ga čuje za d/c sekundi:

      TDOA_max = d / c = 0.05 / 343 = 145.77 µs

  U uzorcima:
      TDOA_max / T = 145.77 / 62.5 = 2.33 uzorka

  → Nikad ne može biti |n1 − n2| > 2 za ovaj postav.
    Ako jest — lažni trigger, odbaci frame.

----------------------------------------------------------------
2. DISKRETNI TDOA KORACI I KUTEVI
----------------------------------------------------------------

  TDOA(Δn) = Δn × 62.5 µs − 0.9 µs     (Δn = n1 − n2)
  φ(Δn)   = arcsin( TDOA(Δn) × c / d )

  Δn |  TDOA (µs) | omjer (TDOA×c/d) |    φ     | korak do sljedećeg
  ---+------------+-----------------+----------+--------------------
  −2 |  −125.9    |    −0.8638      |  −59.8°  |   —
  −1 |   −63.4    |    −0.4349      |  −25.8°  |  +34.0°
   0 |    −0.9    |    −0.0062      |   −0.4°  |  +25.4°
  +1 |   +61.6    |    +0.4226      |  +25.0°  |  +25.4°
  +2 |  +124.1    |    +0.8513      |  +58.4°  |  +33.4°

  UKUPNO: 5 diskretnih kutnih razreda od −60° do +58°.

----------------------------------------------------------------
3. ZAŠTO RAZLUČIVOST NIJE JEDNOLIKA (arcsin nelinearnost)
----------------------------------------------------------------

  Kutna razlučivost po jednom uzorku TDOA ovisi o trenutnom kutu:

      Δφ ≈ (c / d) × T / cos(φ)

  Što je φ bliže ±90°, to je cos(φ) manji → korak Δφ veći.
  Razlučivost se pogoršava prema rubovima, bolja je oko 0°.

  Pri φ =  0°:  Δφ = (343/0.05) × 62.5e−6 / cos( 0°) ≈ 24.6°
  Pri φ = 30°:  Δφ = (343/0.05) × 62.5e−6 / cos(30°) ≈ 28.4°
  Pri φ = 60°:  Δφ = (343/0.05) × 62.5e−6 / cos(60°) ≈ 49.1°

  Vizualno (nejednoliki razmaci zbog arcsin):

  −90°   −60°         −26°   −0.4°   +25°         +58°   +90°
    |      |            |      |       |             |      |
    ├──────┼────────────┼──────┼───────┼─────────────┼──────┤
         Δn=−2        Δn=−1  Δn=0   Δn=+1         Δn=+2
         (razmaci prema rubovima su širi — arcsin rastezanje)

----------------------------------------------------------------
4. KAKO POVEĆATI RAZLUČIVOST
----------------------------------------------------------------

  a) Povećati razmak mikrofona d:
     Veći d → više uzoraka u TDOA_max → više kutnih razreda.

     d = 10 cm → TDOA_max = 291.5 µs = 4.66 uzoraka →  9 razreda
     d = 20 cm → TDOA_max = 583.1 µs = 9.33 uzoraka → 18 razreda

     Ograničenje: veći d → prostorna alijasin za visoke frekvencije
     (d > λ/2, gdje λ = c/f_signal), i fizički veći uređaj.

  b) Povećati frekvenciju uzorkovanja fs:
     Manji T → finija kvantizacija TDOA.

     fs = 48 kHz → T = 20.8 µs → TDOA_max = 7.0 uzoraka → 14 razreda
     fs = 96 kHz → T = 10.4 µs → TDOA_max = 14.0 uzoraka → 28 razreda

  c) Subsample interpolacija (zahtijeva bolji algoritam od threshold):
     GCC-PHAT ili parabolična interpolacija daju procjenu TDOA
     s preciznošću manjom od jednog uzorka → praktički kontinuirani kut.

----------------------------------------------------------------
5. ZAKLJUČAK ZA TRENUTNI POSTAV (d=5 cm, fs=16 kHz)
----------------------------------------------------------------

  Moguće je razlikovati samo 5 kutnih zona:

      Zona 1:  φ ≈ −60°   (mic2 čuje znatno ranije, Δn = −2)
      Zona 2:  φ ≈ −26°   (mic2 čuje malo ranije,   Δn = −1)
      Zona 3:  φ ≈   0°   (oba čuju istovremeno,    Δn =  0)
      Zona 4:  φ ≈ +25°   (mic1 čuje malo ranije,   Δn = +1)
      Zona 5:  φ ≈ +58°   (mic1 čuje znatno ranije, Δn = +2)

  Za prvu demonstraciju i validaciju algoritma — dovoljno.
  Za finiju razlučivost: povećati fs (48/96 kHz) ili d (10+ cm).
